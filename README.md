# 🚗 AI-Based Autonomous Navigation System

## 📌 Project Overview
This project implements an AI-based autonomous navigation system using the A* path planning algorithm in a grid-based simulation environment. The system enables an agent to navigate from a start point to a goal location while intelligently avoiding obstacles.

---

## 🎯 Problem Statement
Autonomous navigation is a key challenge in robotics and self-driving systems. This project demonstrates how an intelligent agent can:
- Understand an environment  
- Plan an optimal path  
- Avoid obstacles  
- Reach a target efficiently  

---

## 💡 Key Features
- A* path planning algorithm implementation  
- Autonomous navigation in a simulated environment  
- Obstacle avoidance system  
- Grid-based environment modeling  
- Visualization of path and movement  
- Output image generation for proof  

---

## 🛠 Tech Stack
- Python  
- NumPy  
- Matplotlib  
- OpenCV  

---

## ⚙️ System Workflow
1. A grid-based environment is generated  
2. Obstacles are randomly placed in the grid  
3. Start and goal points are defined  
4. A* algorithm calculates the shortest path  
5. The agent navigates step-by-step  
6. The final path is visualized and saved

3. Install dependencies
